1. Создать страницу с формой регистрации пользователя.
    В форме должны быть следующие поля:
    - имя
    - фамилия
    - email
    - пароль
    - повтор пароля
2. Реализовать отправку данных этой формы при помощи AJAX.
3. У формы должна быть валидация полей на JS:
    - все поля обязательные
    - у поля email валидация поля на корректность
3. Реализовать обработку AJAX запроса на php.
    В обработчике нужно:
    1) провести валидацию
     - имя не пустое
    - фамилия не пустое
    - email содержит @
    - пароли совпадают

    2) задать некий массив уже существующих юзеров (получать его из какой-либо базы данных не требуется). В массиве должны присутствовать поля email, id, name.
    3) Провести проверку, есть ли в этом массиве элемент с заполненным юзером емейлом.
    4) Результат проверки должен логироваться в файл в любом формате

    При успешной проверке - форма должна скрываться, а пользователю должно выводиться сообщение об успешной регистрации.
    При неудачной проверке - пользователю должна выводиться ошибка над формой.

На бэкенде необходимо использовать фреймворк Codeigniter 3 (на нем написан проект который необходимо поддерживать) либо другой ему подобный.
Можно использовать любые javascript библиотеки. Дизайн страницы на ваш выбор. Для стилизации страницы использовать [getbootstrap.com] версии 3.4.1 или 4.6.x
Версия PHP 5.6 и выше.
Результаты высылать на почту в виде ссылки на проект в Git либо в виде архива файлов.
